🎯 Optimizador de Precios ML

Dado un margen objetivo, calcula el precio mínimo para cada artículo en los 3 canales (Flex, Full Súper, Colecta). Compara con precio actual y sugiere ajuste.

Cargá primero los artículos arriba (📦 Cargar Artículos) y completá el Precio ML actual de cada uno. Luego presioná Optimizar.
